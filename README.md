# ReceptionAI — prezentační web

Marketingový / prezentační web produktu **ReceptionAI** (chytrá cloudová recepce
pro hotely, penziony a ubytovny).

Toto repo je **oddělené od samotné aplikace** — neobsahuje její zdrojový kód.

## Zadání

Kompletní podklad pro stavbu webu je v **[`PREZENTACE-BRIEF.md`](./PREZENTACE-BRIEF.md)**:
pitch, cílová skupina, funkce, tón, brand barvy, struktura stránek, doporučený
tech stack a první prompt pro AI session.

## Doporučený stack

Astro + Tailwind CSS, hosting Netlify / Vercel / Cloudflare Pages. Viz brief, §8.

## Start (pro AI session)

Otevři session nad tímto repem a použij první prompt z `PREZENTACE-BRIEF.md`, §11.
